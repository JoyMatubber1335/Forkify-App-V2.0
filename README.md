# Forkify-App-V 2.0(Under Dev)

This app is a recipe search and meal planning application that allows users to search for recipes, save them to their personal recipe collection,
and generate shopping lists for the ingredients needed to make their chosen meals. 
The app also provides detailed instructions for each recipe and allows users to adjust the serving sizes to meet their needs.
It uses the Food2Fork API to access a large database of recipes and ingredients, making it a convenient tool for home cooks and food enthusiasts alike.
